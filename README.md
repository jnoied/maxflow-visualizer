# 💧 Edmonds–Karp 최대유량 시각화 프로그램

이 프로그램은 Tkinter 기반으로
에드몬드–카프 알고리즘의 작동 과정을 시각화합니다.

## 🔧 실행 방법 (GitHub Codespaces)
1️⃣ 저장소 오른쪽 상단의 초록색 **Code ▼** 버튼 클릭  
2️⃣ **"Open with Codespaces" → "New codespace"** 선택  
3️⃣ 20~30초 후 가상 환경이 자동 실행됨  
4️⃣ 아래 터미널에 입력:
sudo apt update
sudo apt install -y python3-tk
python3 main.py
5️⃣ Tkinter 창이 실행되면 정점/간선 정보를 입력해 시뮬레이션하세요.
